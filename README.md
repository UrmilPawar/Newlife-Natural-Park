# Newlife-Natural-Park
It is a website for a Natural Park developed with help of HTML ,CSS and Bootstrap.

The Website consists of following pages
### 1. Home Page
- Description of Natural Park
- Services Provided
- Active Projects 
### 2. About-Us 
- Vision of the Park
- Objectives of the Park
### 3. Contact Us page
- Reservation form
